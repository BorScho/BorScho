My focus is on classical machine learning, text embeddings, retrieval-based approaches, model evaluation, reproducible pipelines, API-based inference, and monitoring concepts for delayed ground-truth feedback.

Most of my current professional work is not public because it involves employer-owned code, customer data, and confidential business processes. The repositories here mainly document earlier public prototypes and learning projects.
